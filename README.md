## Event and Venue Management API
Uma API avançada e escalável para gerenciar eventos e locais, construída com tecnologias e princípios modernos. Este projeto foi projetado para oferecer um desempenho robusto, código sustentável e integração perfeita com sistemas existentes.

## Recursos
- Gerenciamento de Eventos: Criação, atualização, exclusão e recuperação de eventos de forma simples.
- Gerenciamento de Locais: Administração de detalhes, disponibilidade e reservas de locais.
- Arquitetura Limpa: Aplicação dos princípios de Clean Architecture para melhor organização e manutenibilidade do código.
- Versionamento de Banco de Dados: Utilização do Flyway para migrações consistentes e rastreáveis de banco de dados.
- Containerização: Construído com Docker para implantação simplificada e escalabilidade.
- Java Moderno: Desenvolvido com Java 17, aproveitando os recursos mais recentes da linguagem.

## Tecnologias Utilizadas
- Java 17: Recursos modernos do Java para um código mais limpo e eficiente.
- Spring Framework: Inclui Spring Boot para desenvolvimento rápido e Spring Data para interação com banco de dados.
- Docker: Aplicação containerizada para facilitar a implantação.
- Flyway: Gerenciamento de migrações de esquema de banco de dados.
- PostgreSQL: Banco de dados confiável e escalável.

## Arquitetura
O projeto segue os princípios da Clean Architecture, garantindo:

- Separação de Responsabilidades: Divisão da aplicação em camadas claras (Domínio, Aplicação, Infraestrutura e Apresentação).
- Testabilidade: Alta cobertura de testes e componentes facilmente testáveis.
- Flexibilidade: Adaptabilidade para novas funcionalidades e integrações.
